Biblioteca de Libros

Aplicación desarrollada como desafío final del curso de C# inicial de Quark Academy. Maneja el alta de distintos tipos de usuarios, controla los permisos para prestar libros de acuerdo a cantidades máximas por categorías. Creación de registros de títulos y ejemplares. Historial de préstamos y devoluciones



Tecnologías utilizadas: C#, .NET 6, IDE Visual Studio 2022. Modelado de clases con Miro
