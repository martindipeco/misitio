Kong el Kongquistador

Juego desarrollado para la Game Jam N°3 de Quark Academy, junto a Nicolás Quintana y Omar Moyano. El desafío consistió en entregar un juego en la consola de Windows con temática retro en el lapso de dos semanas. Sobre el juego: el malvado Kong el Kongquistador presenta tres desafíos consecutivos para poder derrotarlo: el ahorcado, un laberinto y un Donkey Kong.



Tecnologías utilizadas: C# .NET Core 3.1 IDE Visual Studio 2022
