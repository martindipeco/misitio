Farmacia Poison

Sitio y web app desarrollado como entrega del curso "Full Stack Python" del programa Codo-a-Codo. El front-end inlcuye validación de formulario de contacto, diseño responsivo con 3 puntos de corte, animación tipo "slides"




Tecnologías utilizadas: Python, Django y SQLite para el Back End. JavaScript, CSS y HTML para el Front End. IDE VS Code Online
