Inmobiliaria

Aplicación de escritorio desarrollada como desafío final del curso de Java de Argentina Programa en la Universidad de La Punta (San Luis). Maneja altas/bajas de propiedades, propietarios e inquilinos. Búsqueda y registro de propiedades en alquiler. Historial de operaciones



Tecnologías utilizadas: C#, .NET 6, IDE Visual Studio 2022. Modelado de clases con Miro
